LIFE WITHOUT TAXES 2 -- README

Life without taxes 2 is a simple game developed for a class project at ASU. 
It was developed in C# using the XNA game framework.
It uses 3D models that are all unique to this game and basic shaders, cameras, and lighting.
The game consists of 4 levels.

Level 1: You are driving to work, but there are obstacles and crazy drivers to avoid.
Level 2: You made it to work. You are a freelance animal control specialist and you have to tame a wild kangaroo.
Level 3: drive back home from work at night, more obstacles and crazier drivers.
Level 4: defend your house from burglers using a tazer gun.
